# Boostify AI - دليل التشغيل والتهيئة

منصة متكاملة لتقديم وإدارة الخدمات المدعومة بالذكاء الاصطناعي، مبنية بالكامل باستخدام تقنيات الويب الأساسية وخدمات Firebase السحابية.

## ⚙️ أولاً: إعداد مشروع Firebase

1. توجه إلى [تنسيق تحكم Firebase](https://console.firebase.google.com/).
2. أنشئ مشروعاً جديداً باسم `Boostify AI`.
3. قم بتفعيل الخدمات التالية داخل منصة التحكم:
   - **Authentication:** تفعيل تسجيل الدخول بواسطة البريد الإلكتروني و كلمة المرور (Email/Password) وتفعيل تسجيل الدخول بواسطة Google.
   - **Cloud Firestore:** إنشاء قاعدة البيانات واختيار منطقة الخادم القريبة لعملائك (مثال: `europe-west3`).
   - **Cloud Storage:** تفعيل مستودع التخزين لرفع الصور والملفات.
   - **Firebase Hosting:** لتوفير استضافة سريعة وآمنة للموقع.
   - **Cloud Messaging:** لتفعيل الإشعارات اللحظية.

## 🚀 ثانياً: تشغيل المشروع محلياً

لتشغيل المشروع محلياً بطريقة تحاكي خوادم الإنتاج دون مشاكل الـ CORS، نوصي باستخدام خادم محلي خفيف مثل `Live Server` في VS Code أو عبر Node.js:

```bash
# تثبيت خادم محلي سريع عالمياً
npm install -g local-server

# تشغيل الخادم في المجلد الرئيسي للمشروع
